# Добро пожаловать в TwilightBlog

Это первый пост блога.

## Что умеет этот блог

- Посты в формате Markdown
- Страница аниме через Anixart API
- Страница игр через Steam API
- Музыкальный плеер SoundCloud
- Падающие лепестки сакуры 🌸

## Пример кода

```csharp
var blog = new TwilightBlog();
blog.Start();
```
